Todo lo que usted necesita!

Social Office Browser es todo lo que usted necesita si quiere abrir varias pestañas de Whatsapp, Facebook, Messenger, Youtube, u otros servicios que solo permiten tener abierto una pestaña o ventana por cada usuario, cada pestaña funciona como un navegador independiente; por ejemplo puede serle util para servicio al cliente. El proyecto es open source, gratis para descargar y usar, y muy facil de usar. No tiene ninguna restricción, ni obligaciones!
