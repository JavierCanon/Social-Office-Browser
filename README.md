# Social Office Browser #

Social Office is a set free and open source tools (mainly in spanish) for Social Media Marketing, Digital Marketing, Social Managers, Community Managers, 
by Javier Cañon: 
 - EN (http://www.javiercanon.com) 
 - ES (http://www.javiercañon.com)

Browser is a tool for browsing in isolated and private tabs.

## What does it Do? ##

* Browsing like navigator does.
* Open each tab like private and isolated session.
* Open many tabs from same app URL (WA, FB, Youtube, etc.) but differents accounts.
